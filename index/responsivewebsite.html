Responsive Website

In the theme responsive website we were introduced to a term “grids” – the most powerful layout system available in CSS. Technically it is a 2-dimensional system, meaning it can handle both columns and rows, unlike flexbox which is largely a 1-dimensional system.

By sticking on the same given style, we had to create a first website for both computer and mobile screens.

Again, it was a tough task, because it took me some time until I have understood we must use grids in order to make the website responsive. Since I was highly satisfied about the prototype I’ve made, I desired to stick with the idea until the very last minute until I have realised that there is still a hudge lack of knowledge for implementing own ideas. Yet, I have learnt to use CSS-grid and Media queries.
